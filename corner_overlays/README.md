# Optional Corner Overlays

Put optional transparent PNG overlays in this directory when you want extra decorative corner artwork:

- `corner_tl.png` for top-left
- `corner_tr.png` for top-right
- `corner_bl.png` for bottom-left
- `corner_br.png` for bottom-right

A corner-sized transparent PNG is anchored to its matching corner. A full-size `360x360` transparent overlay is also accepted.

If no overlay files exist, the installer leaves the watchface artwork unchanged.

Do not commit private artwork or third-party assets unless you have permission to publish them.
