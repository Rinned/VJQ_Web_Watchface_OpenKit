# Credits

This toolkit was built collaboratively by Rinned and Codex (GPT-5.5).

The project grew out of a real-device investigation into Samsung/Tizen wearable watchface installation, switching, local Web storage, and rendering behavior. The final public package keeps only the reusable workflow, scripts, helper/probe source files, and documentation.

The repository intentionally excludes private watchface artwork, personal screenshots, Samsung certificates, account material, store-downloaded packages, and compiled one-off binaries.
